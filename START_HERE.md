## 🐛 Troubleshooting

### App not opening?
- Check if port 5000 is free
- Try: `http://127.0.0.1:5000`

### PDF not uploading?
- Max size: 10MB
- Only text-based PDFs supported

### Summary not good?
- Switch to AI version (`python app.py`)
- Use longer input text (100+ words)
